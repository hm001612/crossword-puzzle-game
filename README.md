# Crossword Puzzle Generator

An interactive crossword puzzle generator supporting both **English** and **Chinese**, with AI-generated clues, multiple difficulty levels, and print functionality.

**Live Demo:** [https://hm001612.github.io/crossword-puzzle-game/](https://hm001612.github.io/crossword-puzzle-game/)

---

## Features

### English Mode
- **Word Library:** 2,923 common English words (sourced from top 3,000 most-used English words)
- **PEP Primary School Library:** Vocabulary organized by grade (Grades 1–6), supports multi-grade selection
- **AI-Generated Clues:** Every word has a concise, crossword-style English clue generated by AI
- **Difficulty Levels:**
  - Easy — 3–5 letter words, ~8 words per puzzle
  - Medium — 4–8 letter words, ~10 words per puzzle
  - Hard — 6–12 letter words, ~14 words per puzzle

### Chinese Mode
- **11 Thematic Word Libraries:**
  - Animals (动物) · Food (食物) · Idioms (成语) · Poetry (诗词)
  - Places (地名) · Historical Figures (历史名人) · Technology (IT)
  - Medical (医学) · Law (法律) · Finance (财经) · Automotive (汽车)
- **AI-Generated Chinese Clues:** Each word comes with a descriptive Chinese hint
- **Connectivity-Aware Generation:** Smart algorithm selects words that share characters, maximizing grid density
- **Difficulty Levels:**
  - Easy — 2–3 character words, ~12 words per puzzle
  - Medium — 3–4 character words, ~10 words per puzzle
  - Hard — 4–7 character words, ~8 words per puzzle

### Gameplay
- Click a cell to select it; click again to toggle direction (across ↔ down)
- Type letters directly into the grid
- Arrow keys and Tab to navigate between cells
- Numbered clue list on the left — click any clue to jump to that word

### Toolbar
| Button | Function |
|--------|----------|
| Check Answers | Highlights correct cells green, incorrect red |
| Hint | Reveals a random unfilled cell in yellow |
| Reset | Clears all input |
| Print | Opens a print preview with two options: Student Copy (blank) or Answer Key (filled) |

## How It Works

### Grid Generation Algorithm (`CrosswordGenerator`)
1. Words are sorted by length (longest first)
2. The first word is placed horizontally at the center
3. Each subsequent word finds all possible intersection points with already-placed words
4. The placement with the most crossing letters is selected
5. Collision and adjacency checks prevent overlapping or touching words
6. Grid is trimmed to minimum bounding rectangle and words are numbered top-to-bottom, left-to-right

### Chinese Word Selection (`selectConnectedZhWords`)
Since Chinese characters have lower reuse frequency than English letters, a connectivity-aware algorithm is used:
1. Builds a character → word index map
2. Scores each word by how many other words share at least one character
3. Starts from a high-connectivity seed word
4. Greedily expands by always selecting the next word with the most shared characters with the current set
5. Falls back to random padding if the connected set is insufficient

- **Pure HTML/CSS/JavaScript** — zero dependencies, single file, works offline
